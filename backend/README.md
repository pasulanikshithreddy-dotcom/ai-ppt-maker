# Backend

This folder is reserved for the FastAPI application for AI PPT Maker.

## Suggested Layout

- `app/api/`: route definitions and request handling
- `app/core/`: settings, configuration, and cross-cutting concerns
- `app/services/`: business logic and integrations
- `app/schemas/`: request and response models
- `tests/`: backend tests

Keep the backend responsible for AI orchestration, slide generation, exports, auth, and persistence.
