# Mobile App

This folder is reserved for the React Native Expo application for AI PPT Maker.

## Suggested Layout

- `app/`: Expo Router screens and navigation structure
- `src/components/`: reusable native UI building blocks
- `src/features/`: feature-specific presentation flows
- `src/lib/`: API clients, helpers, and local utilities
- `assets/`: fonts, icons, splash assets, and images

Keep the mobile app focused on quick deck creation, review, and sharing workflows.
